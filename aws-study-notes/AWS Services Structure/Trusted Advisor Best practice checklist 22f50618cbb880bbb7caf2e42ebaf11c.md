# Trusted Advisor: Best practice checklist.

- **What is it? (1-Sentence Pitch):** An automated online tool that provides real-time guidance to help you provision your resources following AWS best practices.
- **Key Features & Concepts:** Provides recommendations across five pillars:
    - **Cost Optimization:** (e.g., idle EC2 instances)
    - **Performance:** (e.g., overutilized EBS volumes)
    - **Security:** (e.g., S3 bucket permissions, MFA on root account)
    - **Fault Tolerance:** (e.g., RDS Multi-AZ, ELB health checks)
    - **Service Limits:** (checking your usage against AWS service quotas)
