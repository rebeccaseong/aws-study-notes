# Athena: Serverless SQL on S3 files.

- **What is it? (1-Sentence Pitch):** An interactive, **serverless** query service that makes it easy to analyze data directly in Amazon S3 using standard SQL.
- **Core Use Cases:** Quickly running **ad-hoc queries** on log files, business reports, or any other data stored in S3 without needing to set up a database or data warehouse.
- **Key Distinction:** Choose **Athena** for serverless, occasional queries on raw S3 data. Choose **Redshift** for a persistent, high-performance, continuously running data warehouse for frequent and complex BI queries.
