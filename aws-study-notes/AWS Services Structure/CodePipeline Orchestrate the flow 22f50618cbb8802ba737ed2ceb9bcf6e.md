# CodePipeline: Orchestrate the flow.

- **What is it? (1-Sentence Pitch):** A fully managed continuous delivery service that orchestrates and automates your entire software release process.
- **Use Case / Distinction:** The "glue" that connects all the other developer tools. It defines the stages of your release pipeline: **Source (CodeCommit) -> Build (CodeBuild) -> Deploy (CodeDeploy)**.
