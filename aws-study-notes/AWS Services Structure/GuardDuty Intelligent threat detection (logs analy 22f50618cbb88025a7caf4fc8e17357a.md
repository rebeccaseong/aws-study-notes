# GuardDuty: Intelligent threat detection (logs analysis).

- **What is it? (1-Sentence Pitch):** An intelligent threat detection service that continuously monitors for malicious activity and unauthorized behavior to protect your AWS accounts.
- **Core Use Cases:** Automatically detecting potential threats like compromised EC2 instances, unusual API calls, or reconnaissance by attackers.
- **How it Works:** It analyzes AWS **CloudTrail logs, VPC Flow Logs, and DNS logs** using machine learning and threat intelligence feeds. It's a detection service, not a prevention service.
