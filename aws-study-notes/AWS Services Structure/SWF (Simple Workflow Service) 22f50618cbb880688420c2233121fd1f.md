# SWF (Simple Workflow Service)

- **What is it? (1-Sentence Pitch):** An older web service for building scalable, resilient applications with both automated and human-powered workflows.
- **Key Distinction:** **Step Functions** is the modern, recommended service for almost all new use cases. SWF is more complex but can be useful for long-running workflows that involve human intervention. For the SAA exam, focus on Step Functions.
